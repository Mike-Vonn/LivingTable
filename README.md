# LivingTable

A persistent world simulation engine with a virtual tabletop presentation layer.

## Phase 1: VTT Battle Map

Self-hosted, open-source VTT accessible over the internet. Multiple campaigns, real-time sync, fog of war, initiative tracking, dice roller.

## Development

```bash
npm install
npm run dev   # server on :3001, client on :5173
```

## Production Deployment

See `deploy/setup.sh` and the deployment checklist in the implementation plan.
